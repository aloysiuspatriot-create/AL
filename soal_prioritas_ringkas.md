## CHAPTER 22: OPTIONS AND CORPORATE FINANCE

### Soal 4: Payoff dan Profit
**Pertanyaan:** Hitung payoff dan profit dari call/put option!

**Rumus Wajib:**
```
Call Payoff = Max(S - K, 0)
Put Payoff = Max(K - S, 0)
Profit = Payoff - Premium
```

**Contoh:**
```
Call: S = $55, K = $50, Premium = $8
Payoff = Max(55 - 50, 0) = $5
Profit = 5 - 8 = -$3
```

### Soal 5: Intrinsic Value vs Time Value
**Pertanyaan:** Hitung intrinsic value dan time value dari option!

**Rumus:**
```
Intrinsic Value (Call) = Max(S - K, 0)
Time Value = Harga Option - Intrinsic Value
```

**Contoh:**
```
S = $45, K = $40, Option Price = $8
Intrinsic = Max(45 - 40, 0) = $5
Time Value = 8 - 5 = $3
```

### Soal 6: Put-Call Parity
**Pertanyaan:** Gunakan put-call parity untuk menghitung harga put option!

**Formula:**
```
C + PV(K) = P + S

Atau: P = C + PV(K) - S
```

**Contoh:**
```
C = $5, S = $60, K = $58, r = 5%
PV(K) = 58 / 1.05 = $55.24
P = 5 + 55.24 - 60 = $0.24
```

## CHAPTER 24: WARRANTS AND CONVERTIBLES

### Soal 7: Warrant vs Call Option
**Pertanyaan:** Jelaskan perbedaan antara warrant dan call option!

**Poin Kunci:**
- Warrant: Diterbitkan perusahaan, saham baru (dilusi), jangka panjang
- Call option: Diterbitkan investor, saham existing (no dilusi), jangka pendek

### Soal 8: Convertible Bonds
**Pertanyaan:** Mengapa perusahaan menerbitkan convertible bonds?

**Jawaban:**
- Kupon lebih rendah (investor dapat equity upside)
- Delayed dilution
- Tax advantage (interest deductible)

### Soal 9: Kapan Convert?

**Aturan:**
```
Convert jika: Conversion Value > Bond Value

Conversion Value = Conversion Ratio Ã— Stock Price
```

**Contoh:**

## CHAPTER 25: DERIVATIVES AND HEDGING RISK

### Soal 10: Forward vs Futures
**Pertanyaan:** Jelaskan perbedaan forward contract dan futures contract!

**Poin Kunci:**
- Forward: OTC, customized, settlement di akhir, counterparty risk tinggi
- Futures: Exchange, standardized, daily mark-to-market, counterparty risk rendah

### Soal 11: Hedging
**Pertanyaan:** Apa itu hedging? Berikan contoh!

**Definisi:** Strategi mengurangi risiko dengan mengambil posisi berlawanan di instrumen derivatif.

**Contoh:**
```
Eksportir akan terima $1 juta dalam 3 bulan.
Khawatir USD melemah.
Solusi: Jual forward USD/IDR untuk lock-in kurs sekarang.
```

### Soal 12: Interest Rate Swap
**Pertanyaan:** Jelaskan mekanisme interest rate swap!

**Jawaban:**
- Perjanjian tukar pembayaran bunga antara 2 pihak
- Pihak A bayar fixed rate â†’ Pihak B bayar floating rate
- Principal TIDAK ditukar (hanya net interest)
- Tujuan: Hedging interest rate risk, mengubah struktur debt

## CHAPTER 29: MERGERS, ACQUISITIONS, AND DIVESTITURES

### Soal 13: Definisi M&A
**Pertanyaan:** Jelaskan perbedaan merger, acquisition, dan takeover!

**Jawaban:**
- Merger: A + B = C (entitas baru terbentuk)
- Acquisition: A beli B, B jadi subsidiary A
- Takeover: Pengambilalihan kontrol (bisa friendly atau hostile)

### Soal 14: Synergy
**Pertanyaan:** Apa itu synergy dan sebutkan jenisnya!

**Definisi:** Value(A+B) > Value(A) + Value(B) â†’ "1 + 1 = 3"

**Jenis:**
1. Operating Synergy:
   - Revenue enhancement (cross-selling, market expansion)
   - Cost reduction (economies of scale, eliminate duplikasi)

2. Financial Synergy:
   - Lower cost of capital
   - Tax benefits
   - Debt capacity meningkat

**NPV Akuisisi:**
```
NPV = Synergy - Premium
```

### Soal 15: Takeover Defense
**Pertanyaan:** Sebutkan mekanisme pertahanan terhadap hostile takeover!

**Jawaban:**
1. Poison Pill - Pemegang saham lama bisa beli saham dengan diskon besar
2. White Knight - Cari acquirer lain yang lebih friendly
3. Staggered Board - Board dipilih bertahap, tidak bisa diganti sekaligus
4. Golden Parachute - Kompensasi besar untuk executives jika change of control

## CHAPTER 30: FINANCIAL DISTRESS

### Soal 16: Financial Distress vs Bankruptcy
**Pertanyaan:** Jelaskan perbedaan financial distress dan bankruptcy!

**Jawaban:**
- Financial Distress: Kondisi ekonomi (kesulitan bayar kewajiban), belum formal
- Bankruptcy: Status hukum formal melalui pengadilan

### Soal 17: Costs of Financial Distress
**Pertanyaan:** Jelaskan direct costs dan indirect costs of financial distress!

**Direct Costs (3-6% nilai aset):**
- Legal fees, accounting fees, court costs, advisory fees

**Indirect Costs (10-20% nilai perusahaan):**
- Lost sales (customers flee)
- Supplier terms memburuk
- Employee turnover
- Fire sale of assets
- Agency costs (risk shifting, underinvestment, asset stripping)

### Soal 18: Chapter 7 vs Chapter 11
**Pertanyaan:** Jelaskan perbedaan Chapter 7 dan Chapter 11!

**Jawaban:**
- Chapter 7 (Liquidation): Jual semua aset, bayar kreditor, perusahaan bubar
- Chapter 11 (Reorganization): Restruktur debt, perusahaan continue operations

**Decision Rule:**
```
Going Concern Value > Liquidation Value â†’ Chapter 11
Going Concern Value < Liquidation Value â†’ Chapter 7
```

## CHAPTER 31: INTERNATIONAL CORPORATE FINANCE

### Soal 19: Three Types of FX Exposure
**Pertanyaan:** Sebutkan dan jelaskan 3 jenis foreign exchange risk!

**Jawaban:**

1. **Transaction Exposure**
   - Risiko dari transaksi yang sudah committed dalam foreign currency
   - Short-term (hari-bulan)
   - Contoh: Account receivable USD akan dibayar 90 hari lagi

2. **Translation Exposure**
   - Risiko dari translasi laporan keuangan subsidiary asing
   - Periodic (accounting effect, bukan cash flow)
   - Contoh: Nilai aset subsidiary berubah di balance sheet karena FX movement

3. **Economic Exposure**
   - Risiko terhadap future cash flows dan competitive position
   - Long-term
   - Contoh: Rupiah menguat â†’ produk Indonesia jadi lebih mahal untuk ekspor

### Soal 20: Hedging FX Risk
**Pertanyaan:** Bagaimana cara hedging risiko nilai tukar?

**Financial Hedging:**
- Forward contracts (lock-in rate)
- Currency options (flexibility)
- Currency swaps (long-term)
- Futures contracts (standardized)

**Operational Hedging:**
- Matching (revenue & cost same currency)
- Diversification (multiple markets, multiple sourcing)
- Product differentiation (strong brand)
- Flexible sourcing (switch based on FX)
